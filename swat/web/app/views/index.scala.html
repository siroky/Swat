Swat is a toolkit for rich internet application development in Scala, that brings the power of Scala into the world of JavaScript. It consists of a Scala to JavaScript compiler, runtime that supports execution of the compiled code, dependency manager, RPC via AJAX with help of custom JSON serializers and deserializers and many other features. For more information, visit the Github Repository. Or you may try the compiler right now, just write some Scala code to the left editor and see how it compiles to JavaScript.
